Länkar och Verktyg

Här hittar du användbara resurser och verktyg som hjälper dig i varje steg av jobbsökningen – från träning inför intervjuer till portföljbygge och tekniska tester.


Tekniska intervjuer och problemlösning

  • LeetCode – Träna på kodningsproblem i olika svårighetsnivåer inför tekniska intervjuer.
  • HackerRank – Plattform där du kan öva på allt från algoritmer till SQL, JavaScript och mer.
  • Interview Cake – Lär dig tänka som en ingenjör med guider och lösningsstrategier.
  • CodeSignal – Öva på uppgifter som liknar verkliga kodtester från företag.
  • Guru99 – Samling av vanliga frågor och svar inför tekniska intervjuer.

Frontendträning och praktiska projekt

  • Frontend Mentor – Lös riktiga designutmaningar med HTML/CSS/JS.
  • CSSBattle – Lekfull plattform där du tränar på att skriva så effektiv CSS som möjligt.

CV och personligt brev

  • Novoresume – Enkel och snygg CV-editor med mallar.
  • Canva – Skapa grafiskt tilltalande CV:n och personliga brev med drag-and-drop.
  • Jobbland – Guide och exempel på hur du skriver ett effektivt personligt brev.
  • Sprio – Lär dig använda AI som stöd i din ansökan utan att tappa din personliga ton.

GitHub och portföljbyggande

  • GitHub – Plattform för versionshantering och portföljvisning.
  • Guide till GitHub-profil – Så bygger du upp din profil med rätt struktur.
  • Awesome GitHub Profile READMEs – Exempel på kreativa och informativa READMEs.

Nätverk och referenser

  • Silverlight Research – Hur du väljer och bygger ditt nätverk professionellt.
  • Reddit /r/networking – Tips och berättelser från andra som bygger sina nätverk.

AI-intervjuträning och mockintervjuer

  • interviewing.io – Öva tekniska intervjuer anonymt med riktiga intervjuare.
  • Pramp – Träna på tekniska intervjuer tillsammans med andra kandidater, gratis.
